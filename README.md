# AssetForge

AssetForge adalah AI-powered SaaS Web App yang membantu developer, designer, dan vibe coder membuat design asset transparan siap pakai dari gambar referensi.

User dapat mengupload gambar referensi, lalu AI akan menghasilkan beberapa variasi asset desain seperti PNG transparan dan SVG auto-trace yang bisa langsung digunakan dalam project website, aplikasi, landing page, undangan digital, dan kebutuhan desain lainnya.

## Fitur Utama

- **Reference Image to Design Asset**: Upload gambar referensi dan biarkan AI mendesain variasi baru yang terinspirasi dari referensi tersebut.
- **Transparent Background**: Menghasilkan asset dengan background transparan secara otomatis.
- **Multiple Style Variations**: Menghasilkan 4 variasi style secara otomatis (Flat, Realism, 3D Illustration, Cartoon, Line Art, Minimalist).
- **Public Inspiration Gallery**: Pamerkan hasil karya ke galeri publik dan lihat karya buatan user lain.
- **Output Multi-Format**: Mendukung format PNG transparan dan SVG auto-trace.
